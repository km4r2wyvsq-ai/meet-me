# Meet me monitoring plan

## Add before alpha
- Sentry for frontend and backend errors
- PostHog dashboards for:
  - signup completed
  - joined group
  - first post
  - first message
  - invite created
  - invite accepted

## Key alerts
- login failure spikes
- upload failures
- realtime subscription failures
- elevated report creation volume
- drop in activation score

## Operational review
- daily launch dashboard review
- weekly creator analytics review
- weekly moderation review
