# Meet me closed alpha runbook

## Before inviting users
- finish QA checklist
- set production environment variables
- deploy production build
- verify health endpoint
- verify Supabase tables and policies
- verify analytics and monitoring

## During alpha
- onboard first 20 to 50 users manually
- watch launch dashboard daily
- gather onboarding friction notes
- respond quickly to moderation issues
- review invite conversion and group activity

## Success criteria
- users join at least one group
- users post or message in first session
- invite links convert
- no major auth or content failures
