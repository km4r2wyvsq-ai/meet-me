# Meet me alpha admin setup

## Goal
Prepare one strong admin account for staging and alpha.

## Recommended admin setup
Use one real email and create one admin account first.

After signup:
1. locate the user in Supabase auth/users
2. make sure a row exists in `profiles`
3. set:
   - `role = 'admin'`

## Recommended first admin actions
- create 2 or 3 starter groups
- create first 3 to 5 starter posts
- generate one invite
- verify feedback admin pages
- verify launch dashboard and status page

## Suggested starter groups
- Build in Public
- City Walkers
- Weekend Recipes
